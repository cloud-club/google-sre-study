## Thoughts
- 자동화는 단순히 시간 절약이 아닌, 일관성과 신뢰성을 제공하는 플랫폼이 되어야 한다!
- 자동화의 진화는 수동 작업 → 외부 자동화 → 내부 자동화 → 자율 시스템의 순서로 진행된다
- 자동화는 시스템의 복잡성을 줄이는 것이 목적이지, 단순히 작업을 대체하는 것이 아니다!
- 자동화의 실패는 수동 작업의 실패보다 더 큰 영향을 미칠 수 있다! 바로바로 대응이 어려우니까...
